# HomeSchool'd API


The api for the HomeSchool'd app was created with Node JS and is organized around REST. It uses a database with tables for school logs and accpets GET, POST, PATCH and DELETE requests.  

## API Utilizes 
* Express
* Knex
* PostgresSQL
* Morgan
* CORS.

### Testing
* Mocha
* Supertest
